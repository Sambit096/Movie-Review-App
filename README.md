# Instructions
Create a free account on Mailtrap.io

On the home screen click "Start Testing"

Navigate to `backend/Services/Implementations/MailService.cs` and copy the username and password from MailTrap, as well as your email into the appropriate fields.

In the CLI, run `dotnet add packet System.Net.Mail`

You will have to update a user email in your database to test.

Build, and test the program.
